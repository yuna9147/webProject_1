이벤트(Event)

    정의: 사용자 또는 브라우저에 의해 발생하는 동작/행위
    click, keydown, keyup, mouseover, submit, change, load 등

    이벤트 리스너(Event Listner)
    정의: 특정 이벤트가 발생했을 때 실행될 함수를 감지하고 기다리는 기능
    역할: 이벤트가 발생하면 연결된 함수를 자동으로 실행
    1) 인라인 방식 (HTML 속성에서 직접 호출)
    2) JS에서 addEventListener로 연결

    이벤트 핸들러(Event Handler)
    정의: 이벤트 발생 시 실행되는 함수 자체
    작성 위치: 외부 JS 파일 또는 script 태그 내부

인라인 이벤트 모델

    HTML 요소 내부에 직접 on이벤트명="함수()" 형식으로 이벤트를 작성하는 방식
    가장 오래된 방식이며, 간단한 이벤트 처리에 주로 사용됨
  • green
  • orange
  • purple

고전 이벤트 모델

    HTML 요소가 가지고 있는 이벤트 속성(onclick, onmouseover 등)에 
    JavaScript 코드에서 직접 이벤트 핸들러를 연결하는 방식.
    
    element.onclick = function() { ... } 형태로 작성.
    하나의 이벤트에 하나의 함수만 연결 가능 (기존 이벤트를 덮어씀).
    HTML에 JS 코드를 작성하지 않기 때문에 구조와 로직 분리 가능.

노력은 결코 배신하지 않는다.

표준 이벤트 모델(addEventListener)

     HTML 요소에 여러 개의 이벤트 핸들러를 등록할 수 있음
    이벤트를 분리해서 관리할 수 있어 가독성, 유지보수성이 뛰어남
    선언 방법
    요소.addEventListener("click", function(){} )
    요소.addEventListener("click", () => {} )
