자바스크립트에서의 데이터 입/출력

1. 데이터 출력하는 구문

        1) window.alert("알림창에 출력할 문구");
        2) console.log("콘솔창에 출력할 문구");
        3) document.write("화면상에 출력할문구");
        4) 선택한요소.innerText = "요소에 출력할 문구";
        5) 선택한요소.innerHTML = "요소에 출력할 문구";

1)window.alert("내용")

        - alert("메시지");
        - window 객체의 메서드로, window. 생략 가능하며, 브라우저에서 경고창(알림창)을 띄우는 메서드이다
        - 사용자에게 정보를 전달할 때 사용된다.

2) console.log("콘솔창에 출력할 문구")

3) document.write("내용")

4) innerText

        자바스크립트에서 요소에 작성된 "텍스트 내용"을 읽거나 변경할 때 사용하는 속성이다
        
        - 내용을 읽을 때: HTML 태그는 모두 무시되고, 순수한 텍스트만 반환된다.
        - 내용을 변경할 때 : 입력한 문자열이 그대로 출력되며, 포함된 태그는 문자 그대로 표시된다. (태그 해석X)

테스트1입니다.
안녕하세요?

5)innerHTML

        자바스크립트에서 HTML 요소의 내용을 포함한 전체 구조(태그 + 속성 + 내용)을 읽거나 변경할 수 있는 속성이다.

        - 값을 읽을 때 : 태그와 속성을 포함한 HTML 코드를 문자열로 반환한다.
        - 값을 변경할 때 : 문자열로 작성된  HTML 코드를 실제 요소로 해석하여 브라우저에 표시한다.
        - 태그가 HTML로 해석되므로 동적으로 구조를 추가하거나 변경할 때 자주 사용된다.

테스트2입니다.
안녕하세요?
